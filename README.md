##  Sobre o Projeto

Este projeto é uma página web responsiva criada para celebrar momentos especiais. O objetivo era fugir dos presentes convencionais e criar uma experiência digital única, onde o usuário interage com a página para revelar fotos e datas importantes.

A aplicação utiliza manipulação avançada de DOM para animações, áudio API para o player de música e CSS moderno (Grid/Flexbox) para o layout.

##  Funcionalidades

* **Timeline "Slot Machine":** Uma animação estilo caça-níqueis que rola números aleatórios até parar na data exata do início do relacionamento.
* **Galeria Polaroid Dinâmica:** Geração automática de cards estilo polaroid via JavaScript, calculando meses regressivos a partir de uma data final.
* **Music Player Customizado:** Um player de áudio totalmente estilizado com controles de Play/Pause e barra de progresso interativa.
* **Efeitos Atmosféricos:** Partículas de neve (`snowflakes`) e chuva de rosas/corações criadas dinamicamente via JS.
* **Design Responsivo:** Layout adaptável para desktop e dispositivos móveis.

##  Tecnologias Utilizadas

* **HTML5 Semântico**
* **CSS3** (Variáveis CSS, Animations, Keyframes, Glassmorphism)
* **Vanilla JavaScript** (Sem dependências externas)
* **Google Fonts** (Playfair Display & Inter)
